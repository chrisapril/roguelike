                     FlatlineRL - a 7DRL by flend
                     ----------------------------

Requirements
------------

Microsoft .net v3.5

Controls
--------

        Arrow Keys       -       Move
           vi Keys
       . [dot] / 5       -       Do nothing
        
            f            -       Fire Weapon
            U            -       Use Utility Item

     Weapon / Utility
          t / T          -       Throw

            x            -       Examine / target
        
            F            -       Full screen mode
            S            -       Save the game
            Q            -       Quit the game [deletes save]
            
            ?            -       Help
